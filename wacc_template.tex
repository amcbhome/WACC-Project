\section*{Executive Summary}
This report computes the Weighted Average Cost of Capital (WACC) for
\textbf{{{ COMPANY_NAME }}} following the standard three-step approach 
\textcite{WatsonHead2019}. \\
Key Results:
\begin{itemize}
  \item WACC (Book Values): \textbf{{{ WACC_BV_PCT }}\%}
  \item WACC (Market Values): \textbf{{{ WACC_MV_PCT }}\%}
\end{itemize}
Market-value WACC is preferred because it reflects current required 
returns of investors.
